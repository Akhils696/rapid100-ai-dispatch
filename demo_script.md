# RAPID-100 Demo Script

## 2-Minute Demo Overview

### Introduction (15 seconds)
"Welcome to RAPID-100, the Real-time AI for Priority Incident Dispatch system. This system transforms emergency call handling by providing instant transcription, classification, and routing recommendations with full explainability."

### Live Demo (90 seconds)

#### Step 1: Show Dashboard Interface (20 seconds)
- Point to the dispatcher dashboard with its professional dark mode UI
- Highlight the clean layout with panels for live calls, summaries, routing, and explanations
- Mention the real emergency dispatch aesthetic

#### Step 2: Simulate Medical Emergency (25 seconds)
- Click "Medical Emergency" button
- Show real-time transcription appearing: "Help! My wife is unconscious and not breathing..."
- Point to emotion intensity meter rising
- Show classification as MEDICAL emergency with CRITICAL severity
- Highlight the extracted location: "123 Main St, Downtown"
- Note the suggested routing to Ambulance Service

#### Step 3: Demonstrate Explainability (20 seconds)
- Point to the explainability panel
- Show highlighted phrases like "unconscious" and "not breathing" that influenced the decision
- Highlight the reasoning: "Person is not responsive, indicating a serious medical emergency"
- Show the confidence score of 90%

#### Step 4: Show AI Decision Timeline (15 seconds)
- Point to the timeline view showing the progression:
  - Speech → Transcript → Classification → Severity → Routing
- Show total processing time under 2.1 seconds
- Emphasize the speed and efficiency

#### Step 5: Human Confirmation (10 seconds)
- Highlight the "AI Suggested – Awaiting Human Confirmation" notice
- Emphasize that this maintains ethical compliance with human oversight

### Impact Summary (15 seconds)
"This system reduces dispatch delays, provides consistent decision-making, and maintains full transparency for accountability. Thank you!"

## Key Talking Points

### Technical Innovation
- Real-time speech-to-text transcription using Whisper
- Multi-service AI pipeline for classification, severity, and location extraction
- Explainable AI with SHAP-inspired explanations
- Sub-3-second end-to-end processing

### Practical Benefits
- Reduces human error in high-stress situations
- Standardizes emergency classification
- Provides audit trails for accountability
- Maintains human-in-the-loop for final decisions

### Safety & Ethics
- Human confirmation required for routing
- Transparent decision-making process
- Full logging for review and improvement
- Designed for real emergency dispatch environments
